# Bài tập lớn Công nghệ Java chủ đề "Chưa đặt tên".
Cấu trúc folder(thêm gì vui lòng sửa vào đây):
```
ChickenInvaders/
│
├── src/
│   ├── game/              # Lớp chính điều khiển game
│   │   ├── Game.java
│   │   ├── GameLoop.java
│   │   └── GameManager.java
│   │
│   ├── entities/          # Các đối tượng trong game
│   │   ├── Player.java
│   │   ├── Enemy.java
│   │   ├── Bullet.java
│   │   └── PowerUp.java
│   │
│   ├── graphics/          # Xử lý đồ họa
│   │   ├── Sprite.java
│   │   └── Animation.java
│   │
│   ├── utils/             # Các tiện ích (helper)
│   │   ├── Constants.java
│   │   ├── Utils.java
│   │   └── SoundPlayer.java
│   │
│   └── input/             # Xử lý đầu vào từ người chơi
│       ├── InputManager.java
│       └── KeyHandler.java
│
├── assets/                # Tài nguyên
│   ├── images/            # Hình ảnh, sprite sheet
│   ├── sounds/            # Âm thanh
│   └── fonts/             # Font chữ
│
├── tests/                 # Unit test
│
└── README.md              # Tài liệu mô tả dự án
```
